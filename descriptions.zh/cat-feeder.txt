* *tag-reader* это неблокирующий input(вход) XBus, подключенный к считывателю RFID-меток.
* *food-a* и *food-b* это simple outputs подключенные к дозаторам корма для домашних животных.
* Автоматическая кормушка для домашних животных может быть настроена для использования максимум тремя домашними животными с установленными на воротнике RFID-метками *tags* которые хранят значение 1, 2, или 3. Для каждого домашнего животного пользователь может указать минимальное количество единиц времени между кормлениями (TIME), количество *food* для выдачи (SIZE), и какой тип *food* для выдачи (TYPE).
* Настройки конфигурации задаются пользователем с помощью комбинации циферблатов, которые могут быть считаны как XBus inputs(входы), и переключателей, которые могут быть считаны как simple inputs.
