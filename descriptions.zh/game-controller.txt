* *x* и *y* это simple inputs подключенные к джойстику.
* *a* и *b* это simple inputs подключенные к кнопкам.
* *radio-rx* это неблокирующий input(вход) XBus, подключенный к радиоприемнику.
* *radio-tx* это output(выход ) XBus, подключенный к радиопередатчику.
* При получении пакета данных по радио от игровой консоли ответьте следующим 3-значным пакетом данных, убедившись, что используются самые последние входные значения:

#############

|     |     | 0, if *a* = 0 and *b* = 0
| *x* | *y* | 1, if *a* = 100 and *b* = 0
|     |     | 2, if *a* = 0 and *b* = 100
|     |     | 3, if *a* = 100 and *b* = 100
